## Getting Started

1. Clone from Github

```
git clone https://github.com/gauravk030/gaurav-knowledge-react-app.git
```

2. Install packages

```
yarn
```

3. Install yarn package

```
npm install -g yarn
```

4. Run project

```
yarn run start
```

5. Error

